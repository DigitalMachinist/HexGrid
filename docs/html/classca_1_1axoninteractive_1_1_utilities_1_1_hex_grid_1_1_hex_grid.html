Detailed Description

A logical hex-grid implementation based on Amit Patel's examples at http://www.redblobgames.com/grids/hexagons/. Creating a new hex grid does not create an array in memory or any other concrete data. It provides a service to query for data such as what 2D point at which a given hex index appears at or which hex a given 2D point is over.
